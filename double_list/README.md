# double_list
Basic operation of double linked list

### 文件结构
- double_list.h: 函数头文件
- double_list.c: 函数实现代码
- test_double_list.c: 函数的测试代码
- Makefile: linux下的编译脚本

- 插入尾结点

- 修改链表中指定的第一个数据
